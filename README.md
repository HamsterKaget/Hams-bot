prefix : '!','?','h!'

##Available Feature
* !ping =>  Conection Test 
* !info bot => Send Embed About This Bot
* !info user => Send Embed About User
* !info user <@mention> => Send Embed About User in Mention
* ! invite => Send Link to invite the bot
